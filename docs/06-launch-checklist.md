# 6. Launch Checklist & Marketing

## Step 1 — Set up the agent (OpenClaw)

- [x] Documentation final
- [x] 5 selling skills + PDF Maker helper written
- [ ] Install the 6 PassIt skills into the OpenClaw agent (folder structure in `skills/`)
- [ ] Install OKX's official marketplace skills — follow the README at [github.com/okx/onchainos-skills](https://github.com/okx/onchainos-skills)
- [ ] One-time capability test: ask the agent to "create a small PDF file" — confirms PDF-first delivery works

## Step 2 — Create the assets

- [ ] Avatar: green checkmark + document, clear at small size
- [ ] Banner: tagline on green / dark navy / highlighter yellow
- [ ] 2–3 sample outputs to show off (e.g. a "Photosynthesis" Quick Reviewer PDF, a 10-question mock exam) — these double as marketing material

## Step 3 — Register & list on OKX.AI

- [ ] Register PassIt: name **PassIt**, category **Lifestyle**, avatar, the locked description (see [01-overview.md](01-overview.md))
- [ ] List the 5 services using the professional A2A listings (see [02-services-and-pricing.md](02-services-and-pricing.md)) — including the Limitations lines

## Step 4 — Test like a buyer (dry run)

- [ ] Order each of the 5 services yourself and check the output against the reviewer structure + size rules
- [ ] Send a messy order (no format chosen, huge topic) — confirm defaults, right-size check, and the one-question rule work
- [ ] Send an order in another language — confirm the reply matches it
- [ ] Confirm every delivery ends with exactly ONE next-step suggestion

## Step 5 — Launch week (marketing)

- [ ] Post the sample reviewers where learners already are: study communities on X, Reddit, Facebook groups, TikTok study content
- [ ] Time pushes around exam seasons (midterms, finals, board exam schedules)
- [ ] Focus on the first 50 sales — on OKX.AI the **score + sold count is the storefront** (the top agent sold 10K+ copies on a 5.0 rating)
- [ ] Ask every happy buyer to leave a review — reputation is on-chain and permanent

## Step 6 — After launch

- [ ] Watch reviews for any accuracy complaint — fix the skill wording the same day (accuracy is our make-or-break)
- [ ] Track which service sells most and lead marketing with it
- [ ] Once sales are steady, revisit the [roadmap](07-roadmap.md) (v2 ideas)
