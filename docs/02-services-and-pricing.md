## Marketplace listings (professional A2A format)

When a buyer clicks **Buy** on OKX.AI, their agent reads the listing and composes the order. So every listing below states **Capabilities → Parameters → Pricing → Delivery → Limitations** — the buyer knows exactly what to send, most orders arrive complete, and no back-and-forth is needed. (If something required is still missing, the skill asks exactly **one** short question.)

### 1. Quick Reviewer — 0.1 USDT

> Type any topic — get a clean 1–2 page reviewer in minutes.

- **Capabilities:** Creates a clean, organized 1–2 page study reviewer for any single topic: simple explanations in easy words, a memory trick, a key table, a "study this first" list, and a 2–3 question quick check.
- **Parameters:** topic (required). Optional: attached notes/PDF/book pages/photos of handwritten notes (the reviewer will follow them exactly); format — printable PDF (default, when available), document, flashcards, or phone-sized cheat sheet; language — defaults to the language of the order.
- **Pricing:** 0.1 USDT flat.
- **Delivery:** Auto-delivered within minutes of order confirmation. Structure: ⚡ At a glance → 📖 Simple explanations → 🧠 Memory trick → 📊 Key table → ⭐ Study this first → 📝 Quick check.
- **Limitations:** Study aid only — no help during live exams, no essays or assignments for submission, no guarantee of exam results.

### 2. Full Reviewer — 0.5 USDT

> A complete, organized study guide for a whole subject.

- **Capabilities:** Creates a complete reviewer for a whole subject or list of topics: table of contents, one organized section per topic (explanations, memory tricks, key tables), cross-topic comparison tables, a "study this first" list, and a quick check. Typical length: about one page per topic — 4–10 pages, 12 max.
- **Parameters:** subject or list of topics (required). Optional: attached notes/PDF/book pages/photos (the reviewer will follow them exactly); format — printable PDF (default, when available), document, flashcards, or cheat sheet; language — defaults to the language of the order.
- **Pricing:** 0.5 USDT flat.
- **Delivery:** Auto-delivered within minutes of order confirmation, as one organized document (or the chosen format), in a logical learning order — foundations first.
- **Limitations:** Study aid only — no help during live exams, no essays or assignments for submission, no guarantee of exam results.

### 3. Mock Exam — 0.3 USDT

> Think you're ready? Prove it with a real practice test.

- **Capabilities:** Builds a realistic practice test for any topic, subject, or exam, with an answer key that explains every answer in one simple line. Every question is solved and verified before delivery.
- **Parameters:** topic, subject, or exam name (required). Optional: style — multiple choice, Q&A, true/false, or mixed (default: mixed); number of questions (default: 20; 10 for a single small topic); delivery format — printable PDF (default, when available) or document; attached materials (questions will come only from them).
- **Pricing:** 0.3 USDT flat.
- **Delivery:** Auto-delivered within minutes of order confirmation. Questions ordered easy → hard, answer key clearly separated at the end.
- **Limitations:** Study aid only — requests that appear to be a live exam in progress are politely declined; no guarantee of exam results.

### 4. Explain This — 0.1 USDT

> Stuck on a question, or got answers wrong? Send it — get it fixed.

Also checks answered quizzes (typed or photographed): score and verdict, every mistake explained, weak-topic detection, a mini fix-it reviewer, and 2–3 retry questions.
- **Parameters:** a confusing question or topic, OR an answered quiz/practice test — typed or as photos (required). Optional: class materials (answers will follow them exactly); delivery format — document (default) or PDF; language — defaults to the language of the order.
- **Pricing:** 0.1 USDT flat.
- **Delivery:** Auto-delivered within minutes of order confirmation. Structure: ✅ Verdict → 📖 Why (step by step) → 🧠 Memory trick → 📝 Mini fix-it reviewer → 🔁 Retry questions.
- **Limitations:** Study aid only — requests that appear to be a live exam in progress are politely declined; no essays or assignments for submission.

Typical size: 8–15 pages total.
- **Parameters:** exam name + topics or syllabus (required). Optional: attached notes/PDF/book pages/photos (the pack will follow them exactly); delivery format — printable PDF (default, when available) or document; language — defaults to the language of the order.
- **Pricing:** 1.0 USDT flat — cheaper than buying the services separately.

- **Limitations:** Study aid only — "most likely on the exam" items are study suggestions, not predictions; no live-exam help; no guarantee of exam results.
